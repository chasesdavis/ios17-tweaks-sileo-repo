# Device Test Tranche 1

Validate the first eight roothide UI tweaks on an iOS 17 Bootstrap device, one package at a time.

## Constraints

- Install one tweak, respring, test, then continue.
- Keep crash logs and syslog excerpts for failures.
- Do not enable all tweaks at once until each passes individually.

## Enough

Each tweak has a pass/fail result, symptoms for failures, and a recommended fix or promotion decision.
