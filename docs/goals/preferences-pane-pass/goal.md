# Preferences Pane Pass

Add minimal on-device preferences to stable tranche 1 tweaks after device testing identifies which knobs are worth exposing.

## Constraints

- Do not add preference panes before device stability is known.
- Prefer a single enable switch plus 1-3 tweak-specific controls.
- Keep defaults matching v1 behavior.

## Enough

The selected stable tweaks have preference bundles or a documented reason to stay zero-config.
